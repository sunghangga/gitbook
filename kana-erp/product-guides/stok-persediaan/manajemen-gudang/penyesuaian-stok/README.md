# Penyesuaian Stok

Penyesuaian stok dilakukan ketika stok aslinya tidak sesuai dengan stok yang ada di sistem dan juga dapat digunakan untuk opening stok dari produk baru dibuat (jika sudah memiliki stok digudang).
